This application is a Python-based network traffic analysis tool designed to monitor and inspect packets traveling across a network interface in real time. Acting as a lightweight packet sniffer, it captures network traffic and analyzes key protocol data to help users better understand communication occurring within a network.

The tool focuses primarily on HTTP and TCP traffic, extracting valuable information such as HTTP headers, request/response details, and payload data from captured packets. By parsing this information, the application provides insights into how data flows between clients and servers.

This project demonstrates a strong understanding of network protocols, packet inspection techniques, and core cybersecurity concepts. It is particularly useful for educational purposes, network debugging, and basic security analysis, allowing users to observe and study real network communications directly from the command line.
